# Design reference screens — Motion & Opal

Заказчик прикладывает 6 скринов в эту папку. Ниже — что на каждом и что из них брать.
Правило общее: берём РАСПОЛОЖЕНИЕ / плотность / поведение, рендерим в визуальной системе Needt
(токены, тонкие границы, близкие поверхности — см. HANDOFF-DESIGN-PASS.md §2). НЕ копируем градиенты/скевоморфизм
Opal и НЕ добавляем чужие фичи.

## 1. Motion — «AI Agenda» → эталон TODAY
- Крупная дата-заголовок («Sun Jul 19»), ниже — обычный **документ**, не карточки.
- «Today's tasks» = простой чеклист: кружок + название + чип даты (`Tu 7/21`) + длительность (`30m`), приглушённые метаданные.
- Справа — узкая колонка дневной **таймлайн** (6 AM–6 PM), спокойная сетка часов.
- Ширина контента ~800px, текст 15px/24px, заголовок ~20px (жирная дата крупнее). БЕЗ рамки вокруг каждой задачи.
- Взять: документ-повестка, приглушённые метаданные, единый язык строк задач, боковой таймлайн на desktop.

## 2. Motion — date picker → эталон DATE PICKER
- Поповер ~443px: слева **календарь** (Mo–Su, текущий месяц, «сегодня 19» помечен красным, выбранная «17» — белый бокс), справа **быстрые даты**: Today / Tomorrow / Next week / In 2 weeks / Next month с уже посчитанными датами.
- Спокойное выбранное состояние, приглушённые даты соседних месяцев.
- Взять: ровно эту раскладку (календарь + быстрые даты), единый компонент; на телефоне — bottom sheet.

## 3. Opal — Timer → эталон FOCUS (главный экран)
- Один крупный **цифровой таймер** сверху, ниже длительность `30m` с круглыми `−/+`, затем одна широкая кнопка **Start**, под ней вторичный чип.
- Взять: иерархию (крупный таймер → длительность ± → одна кнопка Start → вторичное ниже). НЕ брать скевоморфный «будильник» и glow — рендерить плоско в Needt.

## 4. Opal — Sleep Mode → FOCUS (вторичный, только раскладка)
- Две карточки (Bedtime / Wake Up) с дропдаунами, белая кнопка **Enable**, ниже список.
- Взять: спокойные парные карточки-настройки. NB: у Needt НЕТ sleep mode / soundscapes — это только про раскладку, не фичи.

## 5. Opal — My Apps / Rules → FOCUS (только раскладка правил)
- «Blocked Apps» пустое состояние, ряд карточек-правил (Work Time / Weekend Zen, «Starts in 5h/1d»), плитки категорий.
- Взять: вид карточек-правил и пустого состояния. NB: **блокировка приложений в Needt НЕ делается** (только будущее Chrome-расширение) — это чисто визуальный референс, фичу не копировать.

## 6. Opal — «Leave Early?» → FOCUS (bottom sheet подтверждения)
- Таймер идёт (29:53), снизу выезжает **bottom sheet**: иконка, «Leave Early?» + подстрока, кнопки «Hold to Leave» / «Never Mind».
- Взять: паттерн bottom-sheet-подтверждения при досрочном выходе из сессии (в Needt-стиле, без красного градиента).

## Итог по FOCUS (скрины 3–6)
Один крупный цифровой таймер, длительность `−/+`, одна ясная Start, вторичные действия/очередь ниже,
подтверждения через bottom sheet, спокойный тёмный фон. Стиль — Needt (плоско, токены), раскладка — как Opal.

# MOBILE эталон — Tiimo (5 скринов, папка `tiimo-*.png`)
Это ГЛАВНЫЙ референс мобильной версии Needt. Заказчик: мобилка должна остаться ВЕБ/PWA, но выглядеть
как спроектированное приложение (нижняя панель, чистые экраны), НЕ «ужатый десктоп». Полную паритетность
фич на мобилке делать НЕ обязательно — важно, чтобы ключевые экраны работали и выглядели хорошо.
ВАЖНО: Tiimo светлый/тёплый/скруглённый — берём СТРУКТУРУ и раскладку, рендерим в ТЁМНОЙ тёплой системе Needt
(не копировать светлую тему и «блобы»).

- **Bottom tab bar (главное!)** — постоянная нижняя панель: To-do / Today / Focus / Me(или Profile). Плавающая,
  скруглённая, активная вкладка подсвечена (Today = иконка календаря с числом). Это оболочка мобилки.
- **Today (mobile):** крупный заголовок-дата (serif у Tiimo → у нас крупная дата в стиле Needt), пилюля прогресса
  «0/8» слева сверху, `+`/`…` справа. Группы по времени как пилюли (Anytime / Morning / Afternoon) со счётчиком,
  чевроном сворачивания и «+». Задачи — скруглённые карточки-строки: иконка/эмодзи + жирный заголовок +
  длительность + круглый чекбокс; прогресс подзадач (0/4). Пустая группа — пунктирный плейсхолдер.
- **Focus (mobile):** крупный заголовок, диапазон времени, большой круговой таймер, крупное время, одна кнопка
  пауза + «+1 min», сверху пилюли (звук / Start). Совпадает с Opal-направлением (§FOCUS выше).
- **Settings (mobile):** заголовок «Settings», сгруппированные строки-ряды (иконка + подпись + шеврон / значение
  типа «Dark mode»), секции (Settings / Profile / Account). Чисто, без второго описания категории.
- **Me / Insights:** карточки статистики (настроение, стрик планирования) → у нас эквивалент = Focus stats/streaks
  (опционально).

## Мобильный scope (решение заказчика)
Паритет фич на мобилке НЕ требуется. Приоритет мобильных экранов: **Today, Focus, To-do/Tasks, Settings** —
их довести до вида Tiimo и рабочего состояния. Тяжёлые для телефона (Calendar week-grid, Boards, Space) —
упростить под мобилку или показать аккуратную заглушку «best on desktop». Оболочка = нижняя таб-панель.
